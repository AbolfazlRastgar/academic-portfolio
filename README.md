## Academic Path

### Bachelor’s Research: Ant Colony Optimization for Shortest Path Problems

My bachelor’s thesis focused on solving the shortest path problem using the Ant Colony Optimization algorithm.

This work introduced me to metaheuristic algorithms and swarm intelligence. The main idea was inspired by how real ant colonies use indirect communication and pheromone trails to discover efficient paths. In the thesis, this biological behavior was modeled as an optimization algorithm and applied to a graph-based shortest path problem.

Key topics covered:

- Ant Colony Optimization
- Swarm intelligence
- Metaheuristic algorithms
- Graph-based search
- Shortest path problem
- Pheromone update and evaporation
- MATLAB-based simulation
- Cost and fitness evaluation over iterations

This project helped me build a strong foundation in algorithmic thinking, especially in problems where exact solutions are expensive or where adaptive search methods are more practical.

### Master’s Seminar: Security Challenges in TCP/IP-Based Smart Grids

My master’s seminar focused on the security challenges of smart grids that rely on TCP/IP-based communication.

Smart grids introduce two-way communication between power generation, transmission, distribution, service providers, control centers, and end users. This communication improves monitoring, control, reliability, and energy efficiency, but it also exposes critical infrastructure to cybersecurity risks.

The seminar studied security requirements such as:

- Confidentiality
- Integrity
- Availability
- Authentication
- Authorization
- Access control
- Survivability
- Privacy

It also reviewed attacks across different communication layers, including:

- Denial-of-Service attacks
- Distributed Denial-of-Service attacks
- Spoofing
- Man-in-the-Middle attacks
- Replay attacks
- Data injection
- Eavesdropping
- Attacks against smart meters and IoT-based components

This research strengthened my understanding of network security, critical infrastructure, and the relationship between communication architecture and system reliability.

### Master’s Thesis: AI-Based Intrusion Detection in Smart Grids

My master’s thesis focused on designing an intrusion detection system using artificial intelligence tools in smart-grid environments.

The main goal was to improve attack classification accuracy in anomaly-based intrusion detection by combining a Random Forest classifier with a cross-validation-based evaluation approach.

The research was built around the idea that smart grids are large, distributed, and security-sensitive systems. Because cyberattacks against these systems can affect reliability, availability, and user privacy, intrusion detection systems play an important role in identifying suspicious network behavior.

Key technical areas:

- Intrusion Detection Systems
- Anomaly-based detection
- Supervised machine learning
- Random Forest
- Decision Trees
- Cross-validation
- Hyperparameter evaluation
- Attack classification
- Dataset preprocessing
- Model evaluation metrics

Datasets used:

- NSL-KDD
- CICIDS2018

Attack and security concepts studied:

- DoS
- DDoS
- Probe attacks
- R2L attacks
- U2R attacks
- Malicious vs. legitimate traffic classification
- Network behavior analysis

The thesis compared the proposed model with related research and showed improvement in classification accuracy across evaluation criteria.
